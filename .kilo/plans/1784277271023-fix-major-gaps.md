### Task 5: Facebook Crawler — Playwright

**File:** `backend/legal_radar/crawlers/facebook.py` (new)

**Approach:**
- Playwright + stealth plugin (playwright-stealth) để tránh detect headless
- Không login — chỉ crawl public pages
- Flow: tìm kiếm posts theo keywords trên Facebook public search
- Extract: post text, author (page name), timestamp, URL, engagement (likes/shares/comments count)
- Lấy comments trên mỗi post (top-level only)

**Keywords monitor:**
```
NĐ174: "tin giả", "phạt MXH", "xử phạt mạng xã hội", "nghị định 174"
Tin giả chung: "fake news", "tin sai sự thật", "bóc phốt", "tin đồn"
```

**Dependencies:**
```
pip install playwright playwright-stealth
playwright install chromium
```

**Output format:**
```json
{
  "platform": "facebook",
  "content_type": "post",
  "text": "Nội dung bài viết...",
  "author": "Page Name",
  "url": "https://facebook.com/...",
  "timestamp": "2026-07-17T10:30:00",
  "engagement": {"likes": 150, "shares": 45, "comments": 23},
  "comments": [
    {"text": "Bình luận...", "author": "User A", "timestamp": "..."}
  ]
}
```

**Risks:**
- Facebook có thể detect Playwright → cần stealth plugin
- Rate limit: không crawl quá nhanh (delay 2-5s giữa các request)
- Public search có thể bị giới hạn kết quả

**Depend:** Không

### Task 6: YouTube Crawler — Data API v3

**File:** `backend/legal_radar/crawlers/youtube.py` (new)

**Approach:**
- YouTube Data API v3 (free 10K units/day)
- `search().list()` → tìm videos theo keywords
- `commentThreads().list()` → lấy comments trên mỗi video
- Không cần Playwright — API ổn định, không bị block

**API Key:**
- Tạo tại https://console.cloud.google.com → APIs & Services → Enable YouTube Data API v3
- Set `YOUTUBE_API_KEY` trong `.env`

**Quota cost:**
- search.list = 100 units
- commentThreads.list = 1 unit
- 10K units/ngày → ~100 searches + 10K comment fetches

**Output format:**
```json
{
  "platform": "youtube",
  "content_type": "video",
  "text": "Video title + description",
  "author": "Channel Name",
  "url": "https://youtube.com/watch?v=...",
  "timestamp": "2026-07-17T10:30:00",
  "engagement": {"views": 50000, "likes": 1200, "comments": 340},
  "comments": [
    {"text": "Bình luận...", "author": "User B", "timestamp": "...", "likes": 15}
  ]
}
```

**Dependencies:**
```
pip install google-api-python-client
```

**Depend:** Cần YouTube API key

### Task 7: Crawler Scheduler — On-demand + Auto

**File:** `backend/legal_radar/crawlers/scheduler.py` (new)

**Approach:**
- On-demand: `POST /api/crawl` → chạy crawl ngay, trả kết quả

```python
]
FACEBOOK_PAGES = []  # Public pages to monitor (empty = search by keywords)
YOUTUBE_MAX_RESULTS = 10  # Per search query
```

### Task 8: Crawler → Pipeline Integration

**File:** `backend/legal_radar/crawlers/__init__.py` (new) + `backend/legal_radar/pipeline.py` (update)

**Flow:**
```
crawled_raw.jsonl
    │
    ▼
For each crawled item:
    1. LLM extract (Gemini) → {claim, keywords, subject}
    2. Engine classify → {label, reason, citations}
    3. Source search (dynamic) → {source_label, matched_docs}
    4. Guardrails → validate
    5. Append → runs/queue.jsonl
```

**Mapping crawled → QueueItem:**
```python
QueueItem(
    id=crawled_item["url"],  # dedup key
    comment_id=crawled_item["url"],
    text=crawled_item["text"],
    claim=extracted["claim"],
    keywords=extracted["keywords"],
    nhan=label,
    ly_do=reason,
    nhan_nguon=source_label,
    priority=priority,
    # Extra fields for frontend:
    platform=crawled_item["platform"],
    author=crawled_item["author"],
    url=crawled_item["url"],
    timestamp=crawled_item["timestamp"],
    engagement=crawled_item["engagement"],
)
```
