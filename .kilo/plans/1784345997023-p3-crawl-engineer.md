## Task 1: `cleaner.py` (TẠO MỚI)

**File:** `backend/legal_radar/crawlers/cleaner.py`

**Nhiệm vụ:** Clean raw crawled data trước khi feed vào pipeline. Loại bỏ UI garbage, normalize text, reject post rỗng.

**API:**
```python
def clean_post(raw: dict) -> dict | None:
    """Clean 1 raw post. Return None nếu post nên bị reject."""

def clean_comment(text: str) -> str | None:
    """Clean 1 comment text. Return None nếu là garbage."""

def _is_ui_garbage(text: str) -> bool:
    """Check nếu text là FB UI element."""
```

**UI garbage patterns** (dùng regex, extracted từ `runs/crawled_raw.jsonl` hiện tại):

```python
_UI_PATTERNS = [
    r"^Find friends$",
    r"^View Post$",
    r"^See more$",
    r"^No comments yet$",
    r"^Be the first to comment\.$",
    r"^Photos from .+",
    r"^See translation$",
    r"^Interested$",
    r"^Not interested$",
    r"^Newest\s*$",
    r"^Most relevant$",
    r"^.+?['']s post$",
    r"^https?://\S+$",
    r"^[A-Za-z0-9+/=\s]{30,}$",
    r"^[a-z0-9]{30,}$",
    r"^[\w\d]{40,}$",
    r"^Ads Manager$",
    r"^Facebook$",
    r"^Meta AI$",
    r"^Friends$",
    r"^Memories$",
    r"^Saved$",
    r"^Groups$",
    r"^Reels$",
    r"^Marketplace$",
    r"^Feeds$",
    r"^Events$",
    r"^Privacy\b",
    r"^Are you interested",
]
```

**Rules:**
- Post text < 10 chars → `return None`
- Comment text < 5 chars → skip
- Comment match UI pattern → skip
- Post có 0 comments sau clean → vẫn giữ (post text có thể hữu ích)
- Preserve: `text`, `author`, `url`, `timestamp`, `engagement`, `comments` (cleaned)

## Task 2: `filter.py` (TẠO MỚI)

**File:** `backend/legal_radar/crawlers/filter.py`

**Nhiệm vụ:** Domain relevance filter — chỉ giữ post/comments liên quan tin đồn sáp nhập ĐVHC.

**API:**
```python
def is_relevant(text: str) -> bool:
    """Check text có liên quan domain ĐVHC không."""

def filter_posts(posts: list[dict]) -> list[dict]:
    """Filter list posts, giữ lại post relevant."""
```

**Domain keywords:**
```python
_DVHC_KEYWORDS = [
    "sáp nhập", "gộp tỉnh", "giảm tỉnh", "hợp nhất",
    "đơn vị hành chính", "dvhc", "cấp tỉnh", "cấp xã",
    "tỉnh mới", "thành phố mới", "nối tỉnh",
    "16 tỉnh", "20 tỉnh", "34 tỉnh", "166 xã", "58 xã",
    "bộ nội vụ", "nghị quyết quốc hội",
    "sắp xếp", "quy hoạch tỉnh",
    "giảm số lượng", "tách tỉnh",
]
```

**Rule:** text chứa ≥ `_MIN_KEYWORD_MATCH` (default=2) keywords → relevant. So sánh case-insensitive, không dấu cũng match (dùng normalize approach đơn giản: lower + check substring).

**Scope check:** Check trên post text + tất cả comment texts gộp lại.

### 4b: Thêm `crawl_and_process()` function (sau `crawl_now()`):

```python
def crawl_and_process(
    keywords: list[str] | None = None,
    max_posts: int = 20,
    output_path: str | Path | None = None,
) -> dict[str, Any]:
    """Crawl → clean → filter. Return {crawled, relevant, items}."""
    from .cleaner import clean_post
    from .filter import is_relevant
    
    raw_items = crawl_now(keywords=keywords, max_posts_per_platform=max_posts, output_path=output_path)
    
    relevant_items = []
    for raw in raw_items:
        cleaned = clean_post(raw)
        if not cleaned:
            continue
        
        all_text = cleaned["text"]
        for c in cleaned.get("comments", []):
            all_text += " " + c.get("text", "")
        
        if is_relevant(all_text):
            relevant_items.append(cleaned)
    
    return {
        "crawled": len(raw_items),
        "relevant": len(relevant_items),
        "items": relevant_items,
    }
```

## Task 7: `run_facebook_crawler.py` — update

**File:** `run_facebook_crawler.py`

Không thay đổi credentials. Chỉ cần verify rằng `crawl_facebook()` giờ dùng keywords ĐVHC từ `facebook.py` (đã thay ở Task 3). Không cần sửa thêm vì `FALLBACK_QUERIES` được dùng nội bộ trong `_pw_collect_urls()`.

Nếu muốn override keywords từ runner script, thêm param `keywords` vào `crawl_facebook()` call — nhưng hiện tại function signature đã có `keywords=None` param, và `_pw_collect_urls` dùng `FALLBACK_QUERIES` hardcoded, không dùng param này. **Đây là vấn đề cần sửa:**

Sửa `_pw_collect_urls()` trong `facebook.py` để accept `keywords` param:

```python
def _pw_collect_urls(page, needed: int, queries: list[str] | None = None) -> list[str]:
    urls, seen = [], set()
    for query in (queries or FALLBACK_QUERIES):
        ...
```

Và sửa `crawl_facebook()` để pass keywords:

```python
urls = _pw_collect_urls(page, max_posts * 2, queries=keywords or FALLBACK_QUERIES)
```
